- 个人在学习后django restful framework有想法整理一下文档，写一下自己的使用经验。
- 算是自己的一个学习总结，也是帮助自己仔细的梳理一下学到的东西。
- 算是做一个小白向的教程。
- 自己很菜，如果有写的不正确或者不合适的地方希望大家可以指正。

- 对Django有基本了解，想要学习django restful framework使用的。
